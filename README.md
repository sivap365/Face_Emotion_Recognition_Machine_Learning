# Face Emotion Recognition using Machine Learning

## Overview

This project focuses on building a real-time **Face Emotion Recognition System** using Machine Learning techniques. The system detects human faces from images or webcam input and classifies emotions such as **Happy, Sad, Angry, Surprise, Neutral**, etc.

The goal of this project is to demonstrate how computer vision and deep learning can be combined to interpret human emotions, which can be applied in areas like **human-computer interaction, mental health analysis, and smart surveillance systems**.

---

## Features

* Real-time face detection using webcam
* Emotion classification from facial expressions
* Fast and efficient prediction
* Trained ML model for emotion recognition
* Easy-to-use interface for testing

## 🎯 How It Works

1. The system captures video input from the webcam
2. Detects faces using Haar Cascade Classifier
3. Extracts facial features
4. Feeds the data into the trained model
5. Displays the predicted emotion on screen
